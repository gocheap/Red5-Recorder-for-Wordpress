Red5 Recorder allows wordpress users to easily use Red5 Recorder Player on their website to record Video / Audio Streams as response and embed to website.

Get fully customized Red5 Recorder Player with your own Red5 dedicated server for Website http://red5.svnlabs.com/

== Installation ==

1. Copy red5recorder.php into your wordpress plugins folder, normally located in /wp-content/plugins/

2. Login to Wordpress Admin and activate the plugin


A Red5 Recorder Player can be embedded in a post using a tag of the following form:

[red5recorder:key] 

You can also set global options in the Red5 Recorder option menu.
